## ğŸ›  Installation and Setup

Follow these steps to get the app up and running:

1. **Clone the Repository**
   ```bash

   cd pokedex
   ```
2. **Install Dependencies**
   ```bash
   npm install
   ```
3. **Run the App**
   ```bash
   npm run dev
   ```
4. **View in Browser**
   ```bash
   http://localhost:5173
   ```
